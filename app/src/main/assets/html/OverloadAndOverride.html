🎭 重载 (Overload) - 同一个人，不同的“联系方式”

就好比你有一个朋友 小李：

  • 打电话找他（参数：手机号）。
  • 发邮件找他（参数：邮箱）。
  • 微信找他（参数：微信号）。

👉 方法名相同，但参数不同，就是 重载。

🐶 重写 (Override) - 继承关系里的“改口”

父类 Animal 有个方法 speak()，子类可以改写：


class Animal {
    void speak() {
        System.out.println("动物发出声音");
    }
}

class Dog extends Animal {
    @Override
    void speak() {
        System.out.println("汪汪叫");
    }
}

👉 子类重新实现父类方法，就是 重写。

🌟 一句话记忆

  • 重载：一个人多种联系方式（编译时决定）。
  • 重写：继承改口，运行时听谁的（运行时决定）。

📊 重载与重写的对比表

特性 重载（Overload） 重写（Override）
发生位置 同一个类 子类与父类
方法名 必须相同 必须相同
参数列表 必须不同 必须相同
返回值 可相同或不同 相同或子类
访问修饰符 无限制 不能更严格
异常声明 可以不同 不能更宽泛
多态类型 编译时多态 运行时多态
