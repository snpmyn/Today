Java 逻辑运算符说明

1. && （逻辑与 AND）

当左边表达式为 false 时，右边表达式不再执行（短路）。

if (a != null && a.length() > 0) { ... }

左边 false → 右边不执行
左边 true → 右边执行

2. || （逻辑或 OR）

当左边表达式为 true 时，右边表达式不再执行（短路）。

if (isAdmin || user.isOwner()) { ... }

左边 true → 右边不执行
左边 false → 右边执行

3. ! （逻辑非 NOT）

取反操作，将 true 变 false，false 变 true。

if (!isEmpty) { ... }

4. & （逻辑与 / 位与）

不短路，左右表达式都会执行。用于位运算或需要同时计算两边的逻辑表达式。

if (a != null & a.length() > 0) { ... }

5. | （逻辑或 / 位或）

不短路，左右表达式都会执行。用于位运算或逻辑运算。

if (a == 1 | a == 2) { ... }

6. ^ （异或 XOR）

左右不同为 true，相同为 false，不短路。

boolean result = a ^ b;

逻辑运算优先级

! > && > ||

复杂逻辑建议使用括号明确顺序：

boolean result = (a && b) || (!c && d);
